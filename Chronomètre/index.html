Créez une application permettant d'avoir un chronomètre interactif.

1) SetInterval/SetTimeout

Affichez un chronomètre au format HH:MM:SS. Le chronomètre est accompagné de 4 boutons:

  • Le premier bouton "play" permet d'enclencher le chronomètre. Il est actif lorsque le chrono ne fonctionne pas.
  • Le second bouton "pause" permet d'arrêter le chronomètre. Il est actif quand le chronomètre fonctionne.
  • Le troisième bouton "Reset" permet de réinitialiser le chronomètre. Il est actof quand le chrono n'est pas à 00:00:00.
  • le quatrième bouton "sauver" permet de sauvvegarder le temps dans une liste. Il est activ quand le chronomètre n'est pas à 00:00:00.

2) Evenement

Ajoutez un comportement :

  • Quand vous passez la souris sur le chrono, il se met en pause.
  • Quand vous double clickez sur un des temps sauvegardés le chrono reprend ce temps.
  • Quand vous appuyez sur la touche Enter, le chrono se lance.
